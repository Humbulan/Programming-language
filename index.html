🛠️ Hands‑on Labs

Build tools for real‑world cybersecurity & automation

Practice your skills by building these three essential tools. Each lab reinforces core concepts: variables, loops, lists, functions, and conditionals – all in a cybersecurity context.

🎯 Lab 1: Port Scanner Simulator

What you'll learn: Loops, conditionals, user input – simulate scanning for open ports.

Description: The computer picks a "secret port" (1‑10). You guess the port. The game gives hints (too high/low) until you find it.

🐍 Python Version
import random
secret_port = random.randint(1, 10)
attempts = 0
while True:
    guess = int(input("Guess port (1-10): "))
    attempts += 1
    if guess == secret_port:
        print(f"Port found in {attempts} attempts!")
        break
    elif guess < secret_port:
        print("Too low – try a higher port.")
    else:
        print("Too high – try a lower port.")

📝 Lab 2: Log Parser

What you'll learn: Lists, functions, string manipulation – manage and analyze log entries.

Description: Add log messages, view them, and clear the log – just like a real security analyst.

🐍 Log Parser Functions
log_entries = []
def add_log(entry):
    log_entries.append(entry)
    print(f"Logged: {entry}")
def show_logs():
    for i, entry in enumerate(log_entries, 1):
        print(f"{i}. {entry}")
def clear_logs():
    log_entries.clear()
    print("Log cleared.")
